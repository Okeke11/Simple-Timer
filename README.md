# Glassmorphism Focus Timer

A simple, aesthetic countdown timer built with HTML, CSS, and JavaScript. It features a modern "frosted glass" design and audio alerts.

## Features
* **Custom Inputs:** Set hours, minutes, and seconds independently.
* **Visual Alert:** The interface shakes and turns red when time is up.
* **Audio Alert:** Plays a sound when the timer finishes.
* **Modern UI:** Responsive design with a glassmorphism effect.

## How to Use
1.  Download or save the `timer.html` file.
2.  Double-click the file to open it in your web browser (Chrome, Firefox, Safari, etc.).
3.  Enter your desired time and click **Start**.

## Customization
* **Change Audio:** Replace the link in the `<audio>` tag within the HTML code to use a different mp3 file.
* **Change Colors:** Edit the `background` property in the `body` CSS to change the gradient.
